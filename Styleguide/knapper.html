Knapper

Generelt for knapper

Alle knapper skrives med versaler.

Desuden gælder følgende regler for knapper:


    CSS:
    button {
        cursor: pointer;
    }
    
    button span {
        margin-right: 5px;
    }

Hvis knappen indeholder ikon, skrives det på følgende måde:


    HTML
    <span class="material-icons">NAVN FRA GOOGLE ICONS</span>

Almindelig call to action

Bruges til vigtige knapper, der enten fører videre i en proces eller afslutter en proces.

Bruges på hvide baggrunde.


    HTML:
    <button class="cta-btn">
        <span class="material-icons">add</span>
            KNAPPENS TEKST HER
    </button>

    CSS:
    .cta-btn {
        background-color: #0D76BD;
        width: fit-content;
        color: #FFFFFF;
        text-align: center;
        padding: 7px 15px 7px 15px;
        margin-bottom: 10px;
        font-family: 'roboto';
        font-weight: 500;
        border: none;
        border-radius: 5px;
      
        display: flex;
        align-items: center;
    }

Alternativ call to action

Bruges på samme måde som ovenstående, men på mørke baggrunde


    HTML:
    <button class="cta-btn alt">
        <span class="material-icons">add</span>
            KNAPPEN TEKST HER
    </button>

    CSS:
    .cta-btn.alt {
        color: #000;
    }

Almindelig knap

Bruges til andre typer knapper, som ikke kræver særlig formatering


    HTML: 
    <button class="btn">
        <span class="material-icons">add</span>
            KNAPPENS TEKST HER
    </button>

    CSS:
    .btn {
        background-color: #292929;
        width: fit-content;
        color: #0D76BD;
        text-align: center;
        padding: 7px 15px 7px 15px;
        margin-bottom: 10px;
        font-family: 'roboto';
        font-weight: 500;
        border: 1px solid #757575;
        border-radius: 5px;
      
        display: flex;
        align-items: center;
    }

Tabs-knapper

Bruges til processer som en indikator for progression samt navigation

Den valgte tab-knap gives classen "selected" for at vise blå grafik


    HTML: 
    <div class="tab-btn-container">
        <button class="tab-btn selected">Brief</button>
        <button class="tab-btn">Antenna</button>
        <button class="tab-btn">Drone</button>
    </div>

    CSS: 
    .tab-btn-container {
        width: 300px;
        display: flex;
        box-shadow: 0 3px 3px 0 #1b1b1b;
        margin: 0;
    }
      
    .tab-btn {
        background-color: #292929;
        color: white;
        width: 100px;
        border: none;
        padding: 10px;
    }

    .tab-btn.selected {
        color: #0D76BD;
        border-bottom: solid 2px #0D76BD;
    }
